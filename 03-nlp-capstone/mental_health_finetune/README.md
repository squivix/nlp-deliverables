# Mental Health Advice Bot

This is the repo for the Mental Health Advice Bot capstone project. 

# To setup the application:

- clone the repo
```git clone {repo_url}```
- install the requirements
```pip install -r requirements.txt```
- run the webapp using
```python app.py```
